1. Set Up Ionic Capacitor Project
ionic start TalkMate
cd 
npx cap init YourApp com.example.app


2. Ensure Capacitor is installed:
npm install -g @capacitor/cli

Build your Ionic project
ionic build

4. Install Capacitor Android:
npm install @capacitor/android

3. Add the Android platform:
npx cap add android

4. Sync your app with Capacitor:
npx cap sync




Run on mobile phone:
npx cap init [app name] [app ID]

npx cap init [app name] [app ID]

ionic build
npx cap copy

npx cap open android
